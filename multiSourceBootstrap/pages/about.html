About MultiSource

Events can have quite a few different narratives since media from different countries cover news differently. However, readers usually only get to know one side of a story. To change this, MultiSource was invented. MultiSource is a web-based platform on which users can find news articles covering the latest news. What makes MultiSource different is that it uses more than 30.000 news outlets from all over the world so that it provides the user with news articles written from all different viewpoints. In this way the readers can properly orientate themselves on different events, so that later on they can form their own opinion without being influenced by bias media.

The different articles shown on MultiSource are collected via NewsAPI.
The map used on MultiSource is powered by Mapbox.
